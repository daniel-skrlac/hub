# HUB

A simple product configurator for handmade businesses. Create made-to-order products with variants and personalization, collect the exact details you need from customers.

## What it does

- Build product templates (e.g., rings, candles, bags, paintings)
- Add variants: size, color, material, finish, quantity
- Add personalization fields: engraving text, custom notes
- Configure add-ons: gift wrap, premium materials, rush order
- Auto-calculate price (base + options + add-ons)
- Generate a clean order summary for customer approval

## Who it’s for

- Small handmade brands and solo makers
- Sellers who want fewer back-and-forth messages and fewer mistakes
